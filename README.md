# MOCCA Labs

This repository contains coursework and lab assignments for the Motion Control and Character Animation (MOCCA) course at Peking University, which is based on the open course GAME105.

## Repository Structure

- `MoCCA24-Lab1`  
  Focuses on basic character kinematics, including forward kinematics, inverse kinematics, and simple motion retargeting.

- `MoCCA24-Lab2`  
  Contains assignments related to particle-based or dynamic simulation, with supporting image results and experiment materials.

- `MoCCA25-lab3`  
  Covers motion processing and interactive animation tasks such as motion editing, blending, looping, and interactive character control.

- `MoCCA25-lab4`  
  Explores physics-based character control, including PD control, external balance assistance, and static balance in simulation.
